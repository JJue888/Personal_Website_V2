Rose-Hulman Schedule Planner

For my Introduction to Web Design, I worked with a partner to build a website that would allow a user to generate and edit a schedule for a major at Rose-Hulman. To display a schedule, I pulled data stored in a Firebase collection and used bootstrap components to display the schedule. A user would be able to store an editable coopy of the schedule to a seperate collection in firebase to that would be displayed based on information gathered from using firebase auth to have a user login
